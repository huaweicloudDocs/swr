# 策略语法<a name="swr_01_0073"></a>

如下以SWR服务的“SWR Admin”为例，说明策略语法。

![](figures/策略语法.png)

```
{
    "Version": "1.0",
    "Statement": [
        {
            "Action": [
                "SWR:software:*",
                "SWR:dockerimage:*"
            ],
            "Effect": "Allow"
        }
    ]
}
```
