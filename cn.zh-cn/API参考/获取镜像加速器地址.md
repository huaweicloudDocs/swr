# 获取镜像加速器地址<a name="swr_02_0077"></a>

## 功能介绍<a name="section2053162554118"></a>

获取镜像加速器地址。

>![](public_sys-resources/icon-note.gif) **说明：** 
>目前镜像加速功能支持的区域有华北-北京一、华北-北京四、华东-上海二和华南-广州。
