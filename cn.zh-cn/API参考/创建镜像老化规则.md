## 功能介绍<a name="se03aae4436e64394a95dc13b6f233898"></a>

## 调试<a name="section85822133314"></a>

-   请求样例

    ```
    POST https://{Endpoint}/v2/manage/namespaces/group/repos/busybox/retentions
    ```

    Body：

    ```
    {
        "algorithm": "or",
        "rules": [
            {
                "template": "date_rule",
                "params": {
                    "days": "30"
                },
                "tag_selectors": [
                    {
                        "kind": "label",
                        "pattern": "v5"
                    },
                    {
                        "kind": "label",
                        "pattern": "1.0.1"
                    },
                    {
                        "kind": "regexp",
                        "pattern": "^123$"
                    }
                ]
            }
        ]
    }
    ```
